# SOC Phishing Investigation Project

## Overview
This project simulates a SOC investigation of a phishing attack.

## Scenario
A user clicked a malicious link which triggered PowerShell activity and external communication.

## What I Did
- Analyzed logs (email, endpoint, network)
- Identified attack stages
- Determined system compromise
- Created detection rules
- Simulated SOC alert triage and incident response
  
## Incident Response
This project also includes a simulated SOC response process:
- Alert detection
- Triage and severity classification
- Containment actions
- Eradication steps
- Recovery and lessons learned

## SOC Workflow Demonstrated
Attack → Detection → Triage → Investigation → Response → Lessons Learned
  
## Skills Demonstrated
- Log analysis
- Threat detection
- SOC investigation methodology
- Incident response fundamentals
- Understanding attacker behavior

## Attack Techniques Used
- Phishing (Initial Access)
- PowerShell Execution
- Persistence via Registry
- Command & Control (C2)

## Tools & Concepts
- Log Analysis
- Threat Detection
- SOC Investigation Workflow
