# SOC Phishing Investigation Project

## Overview
This project simulates a SOC investigation of a phishing attack.

## Scenario
A user clicked a malicious link which triggered PowerShell activity and external communication.

## What I Did
- Analyzed logs (email, endpoint, network)
- Identified attack stages
- Determined compromise
- Created detection rules

## Skills Demonstrated
- Log analysis
- Threat detection
- SOC investigation methodology
- Understanding attacker behavior
